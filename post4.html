JavaScript 语法

2.3 操作

操作（operation）：计算和处理数据。

算术操作符

加（+）、减（-）、乘（*）、除（/）

拼接（concatenation）：多个字符串首尾相连在一起

字符串和数值拼接===>更长的字符串=========>"10"+20=====>1020

数值和数值拼接===>两个数值的算术和========>10+20======>30

"+="可以一次完成“加法和赋值”操作。如下：

var year = 2014；

var message = "The year is" ;

message += year;

alert(message);

2.4 条件语句

if后的{}在不影响代码可读性的情况下可省；

2.4.1 比较操作符

比较操作符：>,<,>=,<=,==,===,!=,!==

==不严格相等。eg：false=" ";

===全等，值和类型都相等。

赋值运算并非总是返回真值：if(a = false){alert('hello,world');}中的alert语句不会执行。

2.4.2 逻辑操作符

&&;||;!

2.5 循环语句

2.5.1 while 循环

while循环和if语句的唯一区别：

只要给定条件的求值结果是true，包含在花括号里的代码就将反复地执行下去。

这里的关键是在while循环的内部必须发生一些会影响循环控制条件的事情。从而使循环终止。

do-while循环：某些场合希望循环至少执行一次。

2.5.2 for循环

对某个数组里的全体元素进行遍历处理。

2.6 函数

函数（function）：一组允许在你的代码里随时调用的语句。

在命名变量时，一般用下划线分隔各个单词； 在命名函数时，从第二个单词开始把每个单词的第一个字母写成大写形式，即驼峰命名法。

变量的作用域

全局变量（global variable）：可以在脚本的任意位置被引用。

局部变量（local variable）：只存在于声明它的那个函数的内部。

2.7 对象

对象（object）：自包含的数据集合。包含在对象的数据可以通过两种形式访问——属性（property）和方法（method）：

  • 属性是隶属于某个特定对象的变量；
  • 方法是只有某个特定对象才能调用的函数。

对象就是由一些属性和方法组合在一起而构成的一个数据实体。

在JavaScript里，属性和方法都使用“点”语法来访问。

为了使对象描述一个个体，需要创建这个对象的实例（instance）。实例是对象的具体个体。使用new 关键字。

2.7.1 内建对象

内建对象（native object）：预先定义好的对象。

数组就是一种内建对象。当我们使用new关键字去初始化一个数组时，其实是在创建一个Array对象的新实例：

var beatles = new Array();

当需要了解某个数组有多少元素时，利用Array对象的length属性来获得这一信息：

beatles.length；

Array对象只是诸多JavaScript内建对象中的一种。其他例子包括Math对象和Date对象，它们分别提供了许多非常有用的方法供人们处理数值和日期值。

2.7.2 宿主对象

宿主对象（host object）：由浏览器提供的预定义对象。

宿主对象包括Form、Image和Element等。我们可以通过这些对象获得关于网页上表单、图像和各种表单元素等信息。
